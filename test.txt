<div class="w:1rem h:1rem p:1rem m:1rem"></div>
というhtmlがあった時に
.w\:1rem {
width:1rem;
}
.h\:1rem {
height:1rem;
}
.p\:1rem {
padding:1rem;
}
.m\:1rem {
margin:1rem;
}
というCSSがhtmlを読み込むだけで、自動でstyle.cssに吐き出されるようにしたい
今回1remとなっているが、0.1remから200remまで好きな値を入力した時にその値になるようにしたい。
使われていないクラスは削除したい。
上記の作業をコーポレートサイトの規模でできるようにしたい
